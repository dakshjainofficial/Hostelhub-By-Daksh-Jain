## 🌟 The Problem

Every semester thousands of students face the same problems:

- 📚 Seniors want to sell books after exams.
- 💻 Students want affordable second-hand electronics.
- 🚲 Cycles remain unused after graduation.
- 🪑 Hostel furniture gets abandoned.
- 📦 Students struggle to find trusted local buyers.

Current platforms like OLX and Facebook Marketplace are not designed for college students.

Problems:
- ❌ Trust issues
- ❌ Fake listings
- ❌ Long-distance meetups
- ❌ Scams
- ❌ Irrelevant products

---

## 💡 The Solution

HostelHub solves this by creating a college-exclusive marketplace.

Users can:

- Buy products from students in their own college
- Sell unused items quickly
- Chat directly with buyers and sellers
- Meet safely on campus
- Build trust through verified college communities

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/dakshjainofficial/Hostelhub-By-Daksh-Jain.git
```

## Install Dependencies

```bash
npm install
```

## Create Environment File

```env
PORT=
MONGODB_URI=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

## Run Development Server

```bash
npm run dev
```
